# Manual Mestre do Pesquisador Iniciante

Página web estática criada para apresentar um guia completo sobre pesquisa científica, escrita de artigos e publicação acadêmica.

## Como usar

- Abra `index.html` em qualquer navegador moderno
- O layout é responsivo e compatível com desktops e dispositivos móveis

## Estrutura

- `index.html` — conteúdo principal
- `styles.css` — estilo e identidade visual

## Publicação

O projeto foi pensado para ser hospedado em GitHub Pages como site estático.
